使用说明

快速开始指南

安装与配置

# 克隆仓库
git clone https://github.com/your-org/model-repo.git
cd model-repo

# 安装依赖
pip install -r requirements.txt

快速示例

# 运行CV模型
python examples/cv.py --model yolov8n

# 运行LLM模型
python examples/llm.py --model Qwen3-4B
Build-In 推理

使用内置推理引擎，无需额外服务，适合本地开发和测试。

vLLM 服务

使用vLLM高性能推理服务，支持并发请求，适合生产环境部署。

模型下载

所有模型均提供Hugging Face或官方代码库下载链接。
